• "DRAW" allows the user to add nodes and edges in the canvas with a simple click.
  • "EDIT" allows the user to edit nodes or edges with a simple click on the element. You can edit control points of Bezier Curves, change there colors and move nodes (merging nodes is possible).
  • "ERASER" allows the user to delete a node or an edge by a simple click.
  • "DELETE" reset the canvas".
  • "ERASE" allows to erase nodes or edges with a simple click".
  • "SAVE/LOAD" allows the user to save or load a game.
