• "DRAW" allows the user to add node and edge in canvas by a simple click, furthermore, you can choose the Edge Color with the colored case at the right.
  • "EDIT" allows the user to edit the nodes or the edges, by a simple click on the element, you can edit the control point of the Bezier Curve and change the color of the edge selected, or make a fusion between different nodes.
  • "ERASER" allows the user to delete a node or an edge by a simple click.
  • "DELETE" erase/reset the canvas".
  • "SAVE/LOAD" allows the user to save or to load a game.
