Hackenbush
=========

Hackenbush is a logical game.

The goal is to remove the last edge in the game. (see : [wikipedia] for more details).

Techs
-----------

* [html5] - html5 (canvas)
* [css3] - css3 - Animations, shape,...
* [jQuery] - jQuery
* [php] - To save/load a game
